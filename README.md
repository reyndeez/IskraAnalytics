# Iskra Analytics

Веб-платформа для комплексного мониторинга, аналитики и оценки развития воспитанников хоккейного центра. Проект разработан в качестве выпускной квалификационной работы (диплома).

## Основной функционал

- **Мониторинг показателей:** Сбор и визуализация данных физической подготовки, ледовых тестов и игровой эффективности хоккеистов.
- **Аналитический модуль тренера:** Инструменты для внесения результатов тестирования и отслеживания динамики прогресса подопечных.
- **Интеграция систем:** Полноценное разделение обязанностей между клиентской и серверной частью для обеспечения масштабируемости платформы.

## Технологический стек

- **Backend:** C#, .NET 8 / ASP.NET Core Web API, Entity Framework Core
- **Database:** PostgreSQL
- **Frontend:** React, TypeScript, Tailwind CSS
- **DevOps & Infrastructure:** Docker, Docker Compose, Git

## Архитектурные особенности проекта

В ходе разработки бэкенд-части особое внимание было уделено качеству кода и производительности системы:
- **Слабая связанность (Loose Coupling):** Логика взаимодействия компонентов построена на активном внедрении зависимостей (Dependency Injection) через интерфейсы.
- **Асинхронность:** Все сквозные запросы (от контроллеров до обращений к базе данных через EF Core) реализованы с применением паттерна `async/await` для оптимизации потоков и обеспечения высокой пропускной способности.
- **Контроль ошибок:** Реализована централизованная обработка исключений (`try-catch-finally`), гарантирующая стабильность работы сервиса при некорректных запросах.

## Локальный запуск (Развертывание)

Для локального запуска всей экосистемы проекта (Backend + Frontend + DB) настроена контейнеризация.

1. Убедитесь, что у вас установлен и запущен **Docker Desktop**.
2. Склонируйте репозиторий и перейдите в корневую директорию проекта.
3. Запустите сборку и старт контейнеров одной командой:
   docker-compose up --build
